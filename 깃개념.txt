* 깃허브 계정 만들기
* 환경설정
git config --global user.name "깃허브계정이름"
git config --global user.email "깃허브계정주소"
* 깃 초기화
git init -> 작업 폴더별로 함!


* 작업트리(로컬 : 폴더) -> 스테이지(파일 대기공간) -> 저장소(버전을 만들어 저장하는 곳)
ex) aaa.html파일의 버전 만들기
작업트리에 있는 aaa.html
-> 스테이지에 올리기
-> 저장소에 저장하기 위해 커밋(commit) 명령을 내림
-> 새로운 버전이 생성되면서 스테이지에 대기하던 파일이 모두 저장소에 올라감.
